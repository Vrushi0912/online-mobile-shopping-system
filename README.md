# Online Mobile Shopping System

> A console-based e-commerce application for browsing and purchasing mobile phones, built with C++.

## 📌 Overview

The **Online Mobile Shopping System** is a lightweight, strictly console-based application designed to simulate the core flow of an e-commerce platform. It enables users to browse various mobile phone brands (Redmi, Samsung, Realme), view detailed product specifications and pricing, and calculate a final bill based on selected quantities.

This project demonstrates fundamental C++ programming concepts including flow control, functions, and user interaction.

## ✨ Key Features

- **Brand Selection**: Intuitive menu to choose between major mobile brands.
- **Product Catalog**: Detailed pricing and model listings for each brand.
- **Shopping Cart**: Dynamic quantity selection and real-time total calculation.
- **Interactive UI**: Clean console prompts guiding the user through the shopping process.
- **Robust Input Handling**: Basic validation to handle incorrect menu selections.

## 🛠️ Tech Stack

- **Language**: C++ (Standard 11+)
- **Platform**: Cross-platform console application (Windows/Linux/macOS)
- **Paradigm**: Procedural Programming

## 🚀 Getting Started

### Prerequisites

- A C++ Compiler (GCC, Clang, or MSVC).
- Git (optional, for cloning).

### Compilation & Execution

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/vrushket/online-mobile-shopping-system.git
    cd online-mobile-shopping-system
    ```

2.  **Compile the source code:**
    ```bash
    g++ main.cpp -o shop
    ```

3.  **Run the application:**
    - On Windows:
      ```bash
      ./shop.exe
      ```
    - On Linux/macOS:
      ```bash
      ./shop
      ```
